Issues

A plugin can auto import java jar for Android bp mk project

Features
  • since 2022.3 can use plugin
  • This plugin helps you to develop the Android Framework more easily
  • Plugin has a .bpFile syntax hints and highlighting features
  • if you have set Android Source and build AndroidSource Code success, you can parse the bp file according to libs and staticlibs automatically load jar package
  • Support windows linux mac
  • not support the development of c and c++ but will soon
